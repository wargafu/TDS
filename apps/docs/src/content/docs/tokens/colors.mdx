# Couleurs

Le système de couleurs TDS est construit sur deux niveaux :
**la palette de base** (valeurs brutes) et **les couleurs sémantiques** (significations fonctionnelles).

<Aside type="caution" title="Règle fondamentale">
  Toujours utiliser les **couleurs sémantiques** pour les états fonctionnels (succès, erreur, avertissement, info).
  La palette de base sert à construire des composants, jamais à exprimer directement un état.
</Aside>

---

### Jaune — Couleur nationale, attention

Le jaune est la seconde couleur du drapeau tchadien. Il sert aux avertissements et aux éléments identitaires.

| Token | Hex | Usage |
|---|---|---|
| `yellow.50` | `#FFFBEB` | Fond d'alerte légère |
| `yellow.100` | `#FFF3CC` | Fond badge avertissement |
| `yellow.200` | `#FFE699` | Bordure de notification |
| `yellow.300` | `#FFDA66` | Accent secondaire |
| `yellow.400` | `#FFCF33` | Éléments d'alerte visible |
| **`yellow.500`** | **`#FFD100`** | **Couleur d'avertissement principale** |
| `yellow.600` | `#E6B800` | Hover sur jaune 500 |
| `yellow.700` | `#B58D00` | Texte sur fond jaune clair |
| `yellow.800` | `#846200` | Texte fort sur fond jaune pâle |
| `yellow.900` | `#4D3A00` | Texte haute emphase |

<Aside type="caution" title="Contraste sur fond jaune">
  Le jaune présente des problèmes de contraste avec du texte blanc.
  Utilisez toujours **du texte sombre** (`neutral.800` ou `neutral.900`) sur fond jaune.
  Le contraste blanc/jaune ne respecte pas WCAG AA.
</Aside>

## Couleurs sémantiques

Les couleurs sémantiques expriment une **signification fonctionnelle**. Chaque état a quatre rôles :
`background`, `border`, `text`, `icon`.

### Info

Utilisée pour les messages informatifs neutres, les aides contextuelles.

| Rôle | Variable CSS | Valeur |
|---|---|---|
| background | `--tds-color-semantic-info-background` | `#F2F5FF` (blue.50) |
| border | `--tds-color-semantic-info-border` | `#A6C1FF` (blue.200) |
| text | `--tds-color-semantic-info-text` | `#002063` (blue.700) |
| icon | `--tds-color-semantic-info-icon` | `#0033A0` (blue.500) |

```html
<div style="
  background: var(--tds-color-semantic-info-background);
  border: 1px solid var(--tds-color-semantic-info-border);
  border-left: 4px solid var(--tds-color-semantic-info-icon);
  color: var(--tds-color-semantic-info-text);
  padding: var(--tds-spacing-4);
  border-radius: var(--tds-radius-sm);
">
  Votre demande est en cours de traitement. Délai estimé : 5 jours ouvrables.
</div>
```

### Succès

Confirmations, validations réussies, étapes complétées.

| Rôle | Variable CSS | Valeur |
|---|---|---|
| background | `--tds-color-semantic-success-background` | `#F0FBF5` (green.50) |
| border | `--tds-color-semantic-success-border` | `#A8E9C7` (green.200) |
| text | `--tds-color-semantic-success-text` | `#004224` (green.700) |
| icon | `--tds-color-semantic-success-icon` | `#006B3C` (green.500) |

### Avertissement

Situations nécessitant attention sans être des erreurs.

| Rôle | Variable CSS | Valeur |
|---|---|---|
| background | `--tds-color-semantic-warning-background` | `#FFFBEB` (yellow.50) |
| border | `--tds-color-semantic-warning-border` | `#FFDA66` (yellow.300) |
| text | `--tds-color-semantic-warning-text` | `#0B1220` (neutral.900) |
| icon | `--tds-color-semantic-warning-icon` | `#0B1220` (neutral.900) |

<Aside type="note" title="Texte sombre sur fond jaune">
  L'avertissement utilise `neutral.900` pour le texte et l'icône afin de garantir
  un contraste suffisant sur le fond jaune pâle.
</Aside>

### Danger

Erreurs bloquantes, actions irréversibles, validations échouées.

| Rôle | Variable CSS | Valeur |
|---|---|---|
| background | `--tds-color-semantic-danger-background` | `#FFF2F1` (red.50) |
| border | `--tds-color-semantic-danger-border` | `#FFB0AD` (red.200) |
| text | `--tds-color-semantic-danger-text` | `#82160F` (red.700) |
| icon | `--tds-color-semantic-danger-icon` | `#DA291C` (red.500) |

  textMuted: color.neutral[500],

## Ratios de contraste WCAG

| Combinaison | Ratio | Standard |
|---|---|---|
| `blue.500` sur `neutral.0` | 8.59:1 | ✅ AAA |
| `neutral.900` sur `neutral.0` | 18.1:1 | ✅ AAA |
| `red.700` sur `red.50` | 7.2:1 | ✅ AAA |
| `green.700` sur `green.50` | 6.9:1 | ✅ AA |
| `neutral.900` sur `yellow.50` | 16.4:1 | ✅ AAA |
| `blue.200` sur `neutral.0` | 2.1:1 | ❌ Insuffisant pour texte |
| Blanc sur `yellow.500` | 1.5:1 | ❌ Insuffisant |

<Aside type="danger" title="Combinaisons interdites">
  Ne jamais utiliser `yellow.500` avec du texte blanc, ni `blue.200` comme couleur de texte sur fond blanc.
  Ces combinaisons échouent WCAG AA et rendent le contenu illisible pour les personnes malvoyantes.
</Aside>
