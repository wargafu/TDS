# Espacement

DSTD utilise une grille modulaire basée sur **4px**, exprimée en `rem`.
Chaque token est un multiple de 4px : `spacing-4` = 16px = 1rem.

---

## Principe de la grille 4px

La grille 4px garantit que tous les espacements sont compatibles entre eux.
Combiner `spacing-4` (16px) et `spacing-2` (8px) donne toujours un résultat cohérent.

Les tokens utilisent des nombres entiers qui correspondent au nombre d'unités de 4px :
- `spacing-1` = 1 × 4px = 4px
- `spacing-4` = 4 × 4px = 16px
- `spacing-8` = 8 × 4px = 32px

---

## Groupes d'usage

### Micro (4–12px) — `spacing-1` à `spacing-3`
Espaces internes très petits : icônes et textes, badges, séparateurs fins, padding de tags.

```css
.dstd-badge {
  padding: var(--dstd-spacing-1) var(--dstd-spacing-2);
  /* 4px vertical, 8px horizontal */
}

.dstd-icon-label {
  gap: var(--dstd-spacing-2); /* 8px entre icône et texte */
}
```

### Standard (16–24px) — `spacing-4` à `spacing-6`
Padding de composants, marges internes, espaces entre champs de formulaire.

```css
.dstd-card {
  padding: var(--dstd-spacing-6); /* 24px */
}

.dstd-field + .dstd-field {
  margin-top: var(--dstd-spacing-5); /* 20px */
}
```

### Section (32–56px) — `spacing-8` à `spacing-14`
Espaces entre blocs fonctionnels, séparation de sections, zones d'actions.

```css
.page-section + .page-section {
  margin-top: var(--dstd-spacing-8); /* 32px */
}

.page-actions {
  padding: var(--dstd-spacing-6) var(--dstd-spacing-10);
}
```

### Page (64px+) — `spacing-16` et plus
Marges de page, padding de layout, zones de contenu de grande taille.

```css
.page-container {
  padding: 0 var(--dstd-spacing-6);   /* mobile : 24px */
  max-width: 1024px;
  margin: 0 auto;
}

@media (min-width: 768px) {
  .page-container {
    padding: 0 var(--dstd-spacing-10); /* desktop : 40px */
  }
}
```

---

## Règles de layout

### Mobile
- Padding latéral de page : `spacing-4` (16px) minimum, `spacing-6` (24px) recommandé
- Espace entre sections fonctionnelles : `spacing-6` à `spacing-8` (24–32px)
- Espace interne des cards : `spacing-4` (16px)
- Ne pas dépasser `spacing-10` (40px) entre éléments fonctionnels dans une colonne

### Desktop
- Padding latéral de page : `spacing-10` (40px) ou `spacing-12` (48px)
- Espace entre sections : `spacing-8` à `spacing-12` (32–48px)
- Espace interne des cards : `spacing-5` à `spacing-6` (20–24px)
- Container max-width : 1024px ou 1200px selon le template

### Tableaux et formulaires
- Padding des cellules de tableau : `spacing-4` (16px)
- Espace entre groupes de champs : `spacing-6` (24px)
- Espace entre un label et son champ : `spacing-1` (4px)
- Espace entre champs de formulaire : `spacing-5` (20px)

<Aside type="caution" title="Cibles tactiles">
  Les éléments interactifs (boutons, liens, cases à cocher) doivent avoir une cible tactile
  minimale de **44×44px** pour l'accessibilité mobile (WCAG 2.5.5).
  Combinez `spacing` et `min-height`/`min-width` pour atteindre cette cible.
</Aside>
