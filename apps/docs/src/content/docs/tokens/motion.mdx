# Motion

Le système de motion DSTD est **fonctionnel, jamais décoratif**.
Les animations doivent guider l'utilisateur, pas le distraire.

---

## Durées

| Token | Variable CSS | Valeur | Usage |
|---|---|---|---|
| `instant` | `--dstd-motion-instant` | `0ms` | Reset instantané, suppression d'animation |
| **`fast`** | **`--dstd-motion-fast`** | **`120ms`** | **Hover/focus, micro-interactions** |
| `normal` | `--dstd-motion-normal` | `200ms` | Dropdowns, modales, transitions d'état |
| `slow` | `--dstd-motion-slow` | `320ms` | Panels, toasts, ouvertures |
| `verySlow` | `--dstd-motion-very-slow` | `500ms` | Changements documentaires, page overlays |

---

## Courbes d'easing

| Token | Variable CSS | Valeur | Usage |
|---|---|---|---|
| `standard` | `--dstd-easing-standard` | `cubic-bezier(0.4, 0, 0.2, 1)` | Transitions générales |
| `enter` | `--dstd-easing-enter` | `cubic-bezier(0.3, 0, 0.15, 1)` | Entrées d'éléments |
| `exit` | `--dstd-easing-exit` | `cubic-bezier(0.4, 0, 1, 1)` | Sorties d'éléments |
| `linear` | `--dstd-easing-linear` | `linear` | Loaders, barres de progression |

---

## Combinaisons recommandées

| Composant | Durée | Easing |
|---|---|---|
| Bouton (hover/focus) | `fast` (120ms) | `standard` |
| Dropdown (ouverture) | `normal` (200ms) | `enter` |
| Dropdown (fermeture) | `normal` (200ms) | `exit` |
| Modale (ouverture) | `slow` (320ms) | `enter` |
| Modale (fermeture) | `normal` (200ms) | `exit` |
| Toast (apparition) | `slow` (320ms) | `enter` |
| Tooltip | `fast` (120ms) | `enter` |
| Navigation / page | `normal` (200ms) | `standard` |

---

## Accessibilité — `prefers-reduced-motion`

<Aside type="danger" title="Règle obligatoire">
  Toute animation doit être désactivée lorsque l'utilisateur a activé `prefers-reduced-motion`.
  Ce paramètre indique une sensibilité aux mouvements (épilepsie, vertiges).
</Aside>

```css
/* Pattern obligatoire pour toute transition DSTD */
.mon-composant {
  transition:
    background-color var(--dstd-motion-fast) var(--dstd-easing-standard),
    border-color var(--dstd-motion-fast) var(--dstd-easing-standard);
}

@media (prefers-reduced-motion: reduce) {
  .mon-composant {
    transition: none;
  }
}
```

```css
/* Version utilitaire globale — à inclure dans le reset CSS */
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
    scroll-behavior: auto !important;
  }
}
```

## Variables CSS

```css
--dstd-motion-instant:   0ms;
--dstd-motion-fast:      120ms;
--dstd-motion-normal:    200ms;
--dstd-motion-slow:      320ms;
--dstd-motion-very-slow: 500ms;

--dstd-easing-standard: cubic-bezier(0.4, 0, 0.2, 1);
--dstd-easing-enter:    cubic-bezier(0.3, 0, 0.15, 1);
--dstd-easing-exit:     cubic-bezier(0.4, 0, 1, 1);
--dstd-easing-linear:   linear;
```
