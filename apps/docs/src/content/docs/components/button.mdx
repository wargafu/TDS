# Button

Le bouton DSTD est conçu pour les parcours administratifs : lisible, contrasté, accessible au clavier
et utilisable sans JavaScript.

---

## Importation

```css
/* CSS — requis pour les styles */
@import "@dstd/core/components/button/button.css";
```

```typescript
/* TypeScript — tokens et types */
import { button } from '@dstd/core/components/button';
import type { ButtonVariant, ButtonSize } from '@dstd/core/components/button';
```

---

## Variantes

Cinq variantes couvrent tous les niveaux de hiérarchie d'action.

| Variante | Usage | Quand l'utiliser |
|---|---|---|
| `primary` | Action principale | Valider un formulaire, soumettre une demande |
| `secondary` | Action secondaire | Retour, annuler, action alternative |
| `tertiary` | Action discrète | Voir plus, liens d'interface secondaires |
| `danger` | Action destructive | Supprimer, révoquer, action irréversible |
| `success` | Confirmation positive | Approuver, accepter, finaliser |

## États

| État | Classe / Attribut | Description |
|---|---|---|
| Normal | (aucun) | État par défaut |
| Hover | `:hover` (CSS) | Survol souris |
| Focus | `:focus-visible` (CSS) | Navigation clavier |
| Active | `:active` (CSS) | Clic en cours |
| Disabled | `disabled` + `aria-disabled="true"` | Bouton inactif |
| Loading | `aria-busy="true"` + texte | Traitement en cours |

    ### Composant Button minimal
    ```tsx
    import '@dstd/core/components/button/button.css';
    import type { ButtonVariant, ButtonSize } from '@dstd/core/components/button';

    interface ButtonProps {
      variant?: ButtonVariant;
      size?: ButtonSize;
      disabled?: boolean;
      loading?: boolean;
      type?: 'button' | 'submit' | 'reset';
      onClick?: () => void;
      children: React.ReactNode;
    }

    export function Button({
      variant = 'primary',
      size = 'md',
      disabled = false,
      loading = false,
      type = 'button',
      onClick,
      children,
    }: ButtonProps) {
      const isDisabled = disabled || loading;

      return (
        <button
          type={type}
          className={`dstd-button dstd-button--${variant} dstd-button--${size}`}
          disabled={isDisabled}
          aria-disabled={isDisabled || undefined}
          aria-busy={loading || undefined}
          onClick={onClick}
        >
          {children}
        </button>
      );
    }
    ```

    ### Usage
    ```tsx
    // Formulaire de soumission
    <Button type="submit" variant="primary" size="md">
      Envoyer la demande
    </Button>

    // Action destructive
    <Button variant="danger" onClick={handleDelete}>
      Supprimer le compte
    </Button>

    // État loading
    <Button variant="primary" loading={isSubmitting}>
      {isSubmitting ? 'Envoi en cours…' : 'Soumettre'}
    </Button>

    // Désactivé
    <Button variant="primary" disabled={!isFormValid}>
      Valider (remplissez tous les champs)
    </Button>
    ```

    ### Composant Button
    ```vue
    <!-- components/TdgsButton.vue -->
    <script setup lang="ts">
    import '@dstd/core/components/button/button.css';
    import type { ButtonVariant, ButtonSize } from '@dstd/core/components/button';

    const props = withDefaults(
      defineProps<{
        variant?: ButtonVariant;
        size?: ButtonSize;
        disabled?: boolean;
        loading?: boolean;
        type?: 'button' | 'submit' | 'reset';
      }>(),
      {
        variant: 'primary',
        size: 'md',
        disabled: false,
        loading: false,
        type: 'button',
      }
    );

    const emit = defineEmits<{
      click: [event: MouseEvent];
    }>();

    const isDisabled = computed(() => props.disabled || props.loading);
    </script>

    <template>
      <button
        :type="type"
        :class="`dstd-button dstd-button--${variant} dstd-button--${size}`"
        :disabled="isDisabled"
        :aria-disabled="isDisabled || undefined"
        :aria-busy="loading || undefined"
        @click="emit('click', $event)"
      >
        <slot />
      </button>
    </template>
    ```

    ### Usage
    ```vue
    <TdgsButton type="submit" variant="primary">
      Envoyer
    </TdgsButton>

    <TdgsButton variant="danger" @click="handleDelete">
      Supprimer
    </TdgsButton>

    <TdgsButton :loading="isSubmitting" variant="primary">
      {{ isSubmitting ? 'Envoi…' : 'Soumettre' }}
    </TdgsButton>
    ```

## Accessibilité

<Aside type="tip" title="Règles critiques">
  - Toujours utiliser `<button>` natif (jamais `<div role="button">` ou `<span>`)
  - L'attribut `type` doit être explicite : `type="submit"` ou `type="button"`
  - Un bouton désactivé doit avoir `disabled` ET `aria-disabled="true"`
  - Le label du bouton doit être descriptif : "Supprimer le dossier 2024-001", pas juste "Supprimer"
  - Le focus visible (contour bleu) ne doit jamais être retiré — c'est une exigence WCAG
</Aside>

### Contraste des variantes

| Variante | Fond | Texte | Ratio |
|---|---|---|---|
| primary | `blue.500` (#0033A0) | blanc | 8.59:1 ✅ AAA |
| secondary | blanc | `neutral.800` | 14.2:1 ✅ AAA |
| tertiary | transparent | `neutral.800` | 14.2:1 ✅ AAA |
| danger | `red.500` (#DA291C) | blanc | 5.1:1 ✅ AA |
| success | `green.500` (#006B3C) | blanc | 7.2:1 ✅ AA |
