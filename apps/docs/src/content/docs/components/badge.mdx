# Badge

Les badges indiquent un statut, une catégorie ou un compteur. Ils sont toujours accompagnés
d'un texte visible ou d'un `aria-label` pour l'accessibilité.

## Importation

```css
@import "@tdgs/core/components/badge/badge.css";
```
