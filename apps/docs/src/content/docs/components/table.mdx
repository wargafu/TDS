# Table

Les tableaux DSTD affichent des données structurées : listes de dossiers, résultats de recherche,
données administratives. Ils sont accessibles et responsive par défaut.

## Importation

```css
@import "@dstd/core/components/table/table.css";
```

---

## Modificateurs

```html
<!-- Lignes alternées -->
<table class="dstd-table dstd-table--striped">...</table>

<!-- Avec bordures -->
<table class="dstd-table dstd-table--bordered">...</table>

<!-- Compact (moins de padding) -->
<table class="dstd-table dstd-table--compact">...</table>
```

## Accessibilité

<Aside type="tip" title="Règles critiques">
  - Toujours utiliser `<caption>` pour décrire le contenu du tableau
  - Utiliser `scope="col"` sur les `<th>` d'en-tête de colonne
  - Utiliser `scope="row"` sur les `<th>` d'en-tête de ligne
  - Ne pas utiliser de tableaux pour la mise en page — uniquement pour les données
</Aside>

```html
<!-- Tableau avec en-têtes de ligne et colonne -->
<table class="dstd-table">
  <caption>Tarifs des services administratifs 2024</caption>
  <thead>
    <tr>
      <th scope="col">Service</th>
      <th scope="col">Délai normal</th>
      <th scope="col">Délai urgent</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">Carte d'identité</th>
      <td>15 jours — 2 500 FCFA</td>
      <td>5 jours — 7 500 FCFA</td>
    </tr>
    <tr>
      <th scope="row">Passeport</th>
      <td>30 jours — 25 000 FCFA</td>
      <td>10 jours — 50 000 FCFA</td>
    </tr>
  </tbody>
</table>
```
