# Link

Les liens DSTD sont accessibles, contrastés et cohérents avec la charte institutionnelle.
La classe `.dstd-link` s'applique aux liens de contenu ; les boutons de navigation utilisent `.dstd-button--tertiary`.

## Importation

```css
@import "@dstd/core/components/link/link.css";
```

---

## Variantes

```html
<!-- Lien standard -->
<a href="/services" class="dstd-link">Accéder aux services en ligne</a>

<!-- Lien atténué -->
<a href="/mentions-legales" class="dstd-link dstd-link--muted">Mentions légales</a>

<!-- Lien de danger (confirmation de suppression) -->
<a href="/supprimer" class="dstd-link dstd-link--danger">Supprimer ce dossier</a>

<!-- Lien externe -->
<a href="https://gouv.td" class="dstd-link dstd-link--external" target="_blank" rel="noopener noreferrer">
  Portail officiel du Gouvernement du Tchad
</a>

<!-- Lien standalone (CTA de contenu) -->
<a href="/dossier/001234" class="dstd-link dstd-link--standalone">
  Voir le dossier complet
</a>
```

---

## Lien vs Bouton

| Élément | Quand l'utiliser |
|---|---|
| `<a class="dstd-link">` | Navigation vers une autre page ou ressource |
| `<a class="dstd-button">` | Navigation qui ressemble à une action |
| `<button class="dstd-button">` | Action qui modifie l'état (soumettre, supprimer) |
| `<button class="dstd-button--tertiary">` | Action secondaire discrète dans un formulaire |

---

## Accessibilité

```html
<!-- Lien externe — avertir l'utilisateur -->
<a href="https://..." class="dstd-link dstd-link--external"
   target="_blank" rel="noopener noreferrer"
   aria-label="Portail officiel — s'ouvre dans un nouvel onglet">
  Portail officiel
</a>

<!-- Lien avec contexte supplémentaire pour les AT -->
<a href="/dossier/001234" class="dstd-link">
  Consulter
  <span class="sr-only">le dossier REF-2024-001234</span>
</a>
```
