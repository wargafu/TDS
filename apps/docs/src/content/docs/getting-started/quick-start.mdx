# Démarrage rapide

Ce guide vous permet de construire un formulaire de connexion fonctionnel avec DSTD en 5 minutes.

---

## Prérequis

- `@dstd/core` installé ([voir Installation](../getting-started/installation))
- Un projet web (HTML, React, Vue ou autre)

---

<Steps>

1. **Charger les tokens CSS**

   Ajoutez les imports CSS en tête de votre feuille de style principale.

   ```css
   /* styles/global.css */
   @import "@dstd/core/tokens/color.css";
   @import "@dstd/core/tokens/typography.css";
   @import "@dstd/core/tokens/spacing.css";
   @import "@dstd/core/tokens/radius.css";
   @import "@dstd/core/tokens/shadow.css";
   @import "@dstd/core/tokens/motion.css";

   @import "@dstd/core/components/button/button.css";
   @import "@dstd/core/components/input/input.css";

   /* Polices institutionnelles */
   body {
     font-family: "Source Sans 3", system-ui, sans-serif;
     font-size: var(--dstd-font-size-md);
     line-height: var(--dstd-line-height-md);
     color: var(--dstd-color-neutral-900);
     background-color: var(--dstd-color-neutral-0);
   }
   ```

3. **Afficher un état d'erreur**

   Les champs DSTD gèrent les états d'erreur via des classes modificatrices.

   ```html
   <!-- Champ en erreur -->
   <div class="dstd-field">
     <label class="dstd-field__label" for="email">Adresse e-mail</label>
     <input
       id="email"
       type="email"
       class="dstd-input dstd-input--md dstd-input--error"
       aria-invalid="true"
       aria-describedby="email-error"
       value="adresse-invalide"
     >
     <p id="email-error" class="dstd-error-text" role="alert">
       Veuillez saisir une adresse e-mail valide.
     </p>
   </div>
   ```

4. **Utiliser les tokens TypeScript**

   Accédez aux valeurs de tokens dans votre logique applicative.

   ```typescript
   import { color, spacing, typography } from '@dstd/core';

   // Dans un système de thème, une génération CSS dynamique
   // ou des tests de valeurs
   const palette = {
     primary: color.blue[500],        // '#0033A0'
     danger: color.semantic.danger.text,  // '#82160F'
   };

   const scale = {
     unit: spacing.px[4],   // 16
     large: spacing.px[8],  // 32
   };

   // Vérifier les tailles typographiques
   const sizes = Object.entries(typography.sizes).map(([name, value]) => ({
     name,
     value,
   }));
   ```
