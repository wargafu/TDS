# Installation

`@dstd/core` est disponible sur npm. Choisissez votre gestionnaire de paquets.

## Gestionnaire de paquets

<Tabs>
  <TabItem label="pnpm">
    ```bash
    pnpm add @dstd/core
    ```
  </TabItem>
  <TabItem label="npm">
    ```bash
    npm install @dstd/core
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add @dstd/core
    ```
  </TabItem>
</Tabs>

<Aside type="note" title="Version Node.js">
  DSTD nécessite Node.js 18 ou supérieur. Le package utilise les modules ES natifs (`"type": "module"`).
</Aside>

## Importer les tokens CSS

La manière la plus simple est d'importer les CSS custom properties directement.
Elles sont disponibles pour chaque système de tokens.

```css
/* Importer tous les tokens (recommandé pour démarrer) */
@import "@dstd/core/tokens/color.css";
@import "@dstd/core/tokens/typography.css";
@import "@dstd/core/tokens/spacing.css";
@import "@dstd/core/tokens/radius.css";
@import "@dstd/core/tokens/shadow.css";
@import "@dstd/core/tokens/motion.css";
@import "@dstd/core/tokens/z-index.css";
```

Après import, toutes les variables CSS `--dstd-*` sont disponibles globalement.

```css
/* Utilisation directe */
.mon-composant {
  color: var(--dstd-color-blue-500);
  padding: var(--dstd-spacing-4);
  border-radius: var(--dstd-radius-sm);
}
```

---

## Importer les styles composants

Importez les CSS des composants que vous utilisez.

```css
/* Bouton */
@import "@dstd/core/components/button/button.css";

/* Champs de formulaire */
@import "@dstd/core/components/input/input.css";
```

## Importer les tokens TypeScript

Pour accéder aux valeurs de tokens dans votre code TypeScript/JavaScript :

```typescript
// Importer les tokens individuellement
import { color } from '@dstd/core/tokens';
import { typography } from '@dstd/core/tokens';
import { spacing } from '@dstd/core/tokens';

// Ou tout importer depuis la racine
import { color, typography, spacing, radius, shadow, motion } from '@dstd/core';

// Utiliser les tokens
const primaryBlue = color.blue[500];    // '#0033A0'
const bodySize = typography.sizes.md;   // '1rem'
const gap = spacing.scale[4];           // '1rem' (16px)
```

---

## Importer les types TypeScript

```typescript
import type { ColorTokens } from '@dstd/core/tokens';
import type { ButtonVariant, ButtonSize } from '@dstd/core/components/button';
import type { InputVariant, InputSize, InputState } from '@dstd/core/components/input';
```

## Import JSON (Style Dictionary / Figma)

Les tokens sont aussi disponibles en JSON pour l'intégration avec Style Dictionary,
les plugins Figma ou d'autres outils de design.

```javascript
import colorJson from '@dstd/core/tokens/color.json' assert { type: 'json' };
// ou en CommonJS
const colorJson = require('@dstd/core/tokens/color.json');
```

## Installation sans bundler (CDN / HTML pur)

Pour utiliser DSTD sans outil de build, copiez les fichiers CSS dans votre projet
ou pointez vers un CDN auto-hébergé.

```html
<!doctype html>
<html lang="fr">
<head>
  <!-- Polices institutionnelles -->
  <link rel="preconnect" href="https://fonts.googleapis.com">
  <link href="https://fonts.googleapis.com/css2?family=Source+Sans+3:wght@300;400;500;600;700;800&display=swap" rel="stylesheet">

  <!-- Tokens DSTD (depuis votre serveur) -->
  <link rel="stylesheet" href="/assets/dstd/tokens/color.css">
  <link rel="stylesheet" href="/assets/dstd/tokens/typography.css">
  <link rel="stylesheet" href="/assets/dstd/tokens/spacing.css">
  <link rel="stylesheet" href="/assets/dstd/tokens/radius.css">
  <link rel="stylesheet" href="/assets/dstd/tokens/shadow.css">
  <link rel="stylesheet" href="/assets/dstd/tokens/motion.css">
  <link rel="stylesheet" href="/assets/dstd/tokens/z-index.css">

  <!-- Composants DSTD -->
  <link rel="stylesheet" href="/assets/dstd/components/button/button.css">
  <link rel="stylesheet" href="/assets/dstd/components/input/input.css">
</head>
```

<Aside type="tip" title="Self-hosting des polices">
  En production, hébergez les fichiers WOFF2 de **Source Sans 3** et **Noto Naskh Arabic**
  sur vos propres serveurs. Cela évite les requêtes vers Google Fonts et garantit
  la disponibilité en environnements réseau restreints (intranet gouvernemental).
</Aside>

## Vérification de l'installation

Après installation, vérifiez que les tokens CSS sont bien chargés dans le navigateur :

```javascript
// Dans la console du navigateur
getComputedStyle(document.documentElement)
  .getPropertyValue('--dstd-color-blue-500')
// Doit retourner : ' #0033A0'
```
