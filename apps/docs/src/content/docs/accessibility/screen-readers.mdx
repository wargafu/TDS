# Lecteurs d'écran

Les lecteurs d'écran transforment l'interface visuelle en retour audio ou braille.
DSTD garantit la compatibilité avec les lecteurs d'écran les plus répandus.

---

## Lecteurs d'écran supportés

| Lecteur | Système | Navigateur recommandé |
|---|---|---|
| NVDA | Windows | Firefox |
| JAWS | Windows | Chrome ou Firefox |
| VoiceOver | macOS / iOS | Safari |
| TalkBack | Android | Chrome |
| Orca | Linux | Firefox |

---

## Règles ARIA dans DSTD

### Boutons et actions

```html
<!-- État pressé (toggle) -->
<button type="button" aria-pressed="false">Activer les notifications</button>

<!-- État étendu (accordion, dropdown) -->
<button type="button" aria-expanded="false" aria-controls="menu-content">
  Options
</button>

<!-- Bouton chargement -->
<button type="button" aria-busy="true" aria-label="Envoi en cours, veuillez patienter">
  Envoi en cours…
</button>
```

### Formulaires

```html
<!-- Champ requis -->
<input type="text" required aria-required="true">

<!-- Champ en erreur -->
<input type="text" aria-invalid="true" aria-describedby="champ-erreur">
<p id="champ-erreur" role="alert">Ce champ est requis.</p>

<!-- Aide contextuelle -->
<input type="text" aria-describedby="champ-aide">
<p id="champ-aide">Format attendu : JJ/MM/AAAA</p>
```

### Live regions

```html
<!-- Annonces polies (non urgentes) -->
<div role="status" aria-live="polite" aria-atomic="true">
  Vos modifications ont été enregistrées.
</div>

<!-- Annonces assertives (erreurs critiques) -->
<div role="alert" aria-live="assertive">
  Session expirée. Veuillez vous reconnecter.
</div>
```

## Tester avec VoiceOver (macOS)

1. Activer VoiceOver : `Cmd + F5`
2. Naviguer entre les éléments : `Ctrl + Option + →`
3. Naviguer entre les titres : `Ctrl + Option + Cmd + H`
4. Naviguer entre les formulaires : `Ctrl + Option + Cmd + J`
5. Entrer/sortir d'un groupe : `Ctrl + Option + Shift + ↓` / `↑`

---

## Tester avec NVDA (Windows)

1. Télécharger NVDA : [nvaccess.org](https://www.nvaccess.org/download/)
2. Démarrer NVDA : `Ctrl + Alt + N`
3. Naviguer par titres : `H`
4. Naviguer par formulaires : `F`
5. Naviguer par boutons : `B`
6. Liste de tous les éléments : `NVDA + F7`

<Aside type="tip" title="Test minimal">
  Au minimum, testez chaque nouveau composant avec VoiceOver + Safari sur macOS.
  C'est rapide, gratuit et couvre un large éventail d'utilisateurs.
</Aside>
